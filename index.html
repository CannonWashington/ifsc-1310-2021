Progressive Enhancement

Progressive Enhancement, atleast to me, seems to focus on what the Websites being built for as opposed to the browser it will be used on. This seems like a much better approach to me, as it focuse on what the user is using the website for. It also seems much more reasonable than building for specific more advanced browsers when not everyone uses the same browsers.
